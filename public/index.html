What is this?

REST is a coding architechtural model and comes from the words "Representational State Transfer". It is completely stateless, meaning server side does not store any state about the client session. The state is instead kept and transfered from the client to everywhere. This is the architechture style of the world wide web!

This RESTful API is developed using Nodejs and the MEAN-stack. It implements HTTP services to collect and serve data about different games. This can be used to store data about game time, player behavior and so on for later analysis and other use.
